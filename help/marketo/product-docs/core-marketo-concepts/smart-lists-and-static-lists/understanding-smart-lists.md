# Explicación de las listas inteligentes {#understanding-smart-lists}

Las listas inteligentes le permiten buscar grupos específicos de personas mediante filtros simples. Se utilizan en todo Marketo, como en campañas inteligentes, programas, informes, segmentaciones, modelos empresariales, programas de participación, contenido dinámico y mucho más. Según sus necesidades, se pueden crear listas inteligentes en dos lugares dentro de Marketo:

1. En el **Base de datos**, donde estén disponibles para uso compartido o global (por ejemplo, &quot;Todas las personas sin suscripción&quot;).
1. Entrada **programas** como activos locales, donde encuentran un grupo de personas relevantes para ese programa específico (por ejemplo, &quot;Feria Asistida&quot;).

Los pasos principales para crear una lista inteligente son los siguientes:

1. [Crear una lista inteligente](/help/marketo/product-docs/core-marketo-concepts/smart-lists-and-static-lists/creating-a-smart-list/create-a-smart-list.md)
1. [Buscar y agregar filtros a una lista inteligente](/help/marketo/product-docs/core-marketo-concepts/smart-lists-and-static-lists/creating-a-smart-list/find-and-add-filters-to-a-smart-list.md)
1. [Definición de filtros de listas inteligentes](/help/marketo/product-docs/core-marketo-concepts/smart-lists-and-static-lists/creating-a-smart-list/define-smart-list-filters.md)
