# Usar listas inteligentes integradas/del sistema {#use-built-in-system-smart-lists}

Marketo tiene listas inteligentes integradas muy útiles. ¡Echémosles un vistazo!

>[!NOTE]
>
>Marketo está cambiando términos como Lista negra y Lista blanca a Lista de bloqueados y Lista de permitidos en nuestro producto. Durante esta actualización, es posible que vea los términos antiguos en nuestra interfaz de usuario y en las capturas de pantalla de la documentación, así como los términos nuevos en nuestro texto de documentación. Pedimos disculpas por cualquier confusión.

1. Vaya a la **Base de datos**.

   ![](assets/db.png)

1. Abra el **Listas inteligentes del sistema** carpeta para mostrar la colección.

   ![](assets/two.png)

1. Para ver un ejemplo: Select **Todas las personas** y, a continuación, haga clic en el botón **People** pestaña .

   ![](assets/three.png)

   >[!NOTE]
   >
   >Los filtros de las listas inteligentes del sistema no necesitan aplicarse a cada lista o campaña inteligente. Su contenido se reconoce automáticamente.

A continuación se muestra un resumen de las acciones de las listas inteligentes integradas:

| Nombre de la lista | Descripción |
|---|---|
| Todas las personas | Todas las personas de la base de datos de Marketo |
| Personas que cancelaron su suscripción | Estas personas solo pueden enviarse correos electrónicos operativos; esto suele estar controlado por la propia persona. |
| Marketing suspendido | Estas personas solo pueden enviarse correos electrónicos operativos; esto es controlado por usted, el experto en marketing. |
| Lista de bloqueos | Estas personas no recibirán ningún correo electrónico. |
| Direcciones de emails rechazadas | Personas con una dirección de correo electrónico que no se puede enviar o que rechazan su correo electrónico. |
| Duplicados posibles | Personas que pueden duplicarse en la base de datos de Marketo. |

>[!NOTE]
>
>Las listas inteligentes del sistema no se pueden eliminar. A excepción de la lista Posibles duplicados, tampoco se pueden editar.
