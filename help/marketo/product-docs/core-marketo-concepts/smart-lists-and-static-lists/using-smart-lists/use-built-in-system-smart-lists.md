# Usar Listas inteligentes integradas/del sistema {#use-built-in-system-smart-lists}

Marketing tiene algunas listas inteligentes incorporadas muy útiles. ¡Vamos a echarles un vistazo!

>[!NOTE]
>
>Marketing está en proceso de cambiar términos como Lista negra y Lista blanca a Lista de bloqueados y Lista de permitidos en nuestro producto. Durante esta actualización, puede ver los términos antiguos en nuestra interfaz de usuario y en las capturas de pantalla de documentación, así como los nuevos términos en el texto de la documentación. Pedimos disculpas por cualquier confusión.

1. Vaya a la **Base de datos**.

   ![](assets/db.png)

1. Abra la carpeta **Listas inteligentes del sistema** para mostrar la colección.

   ![](assets/two.png)

1. Para ver un ejemplo: Seleccione **Todas las personas** y haga clic en la ficha **Personas**.

   ![](assets/three.png)

   >[!NOTE]
   >
   >Los filtros de las listas inteligentes del sistema no necesitan aplicarse a cada lista o campaña inteligente. Su contenido se reconoce automáticamente.

A continuación se muestra un resumen de lo que hacen las listas inteligentes integradas:

| Nombre de lista | Descripción |
|---|---|
| Todas las personas | Todas las personas de la base de datos de Marketing |
| Personas no suscritas | Estas personas sólo pueden ser enviadas por correo electrónico operativo; esto suele ser controlado por la propia persona. |
| Comercialización suspendida | Estas personas sólo pueden ser enviadas por correo electrónico operativo; esto generalmente lo controla usted, el especialista en mercadotecnia. |
| Lista de bloqueados | Estas personas no recibirán ningún correo electrónico. |
| Direcciones de correo electrónico devueltas | Personas con una dirección de correo electrónico no entregable o que rechazan su correo electrónico. |
| Duplicados posibles | Personas que podrían duplicarse en la base de datos de Marketing. |

>[!NOTE]
>
>No se pueden eliminar las listas inteligentes del sistema. A excepción de la lista Posibles Duplicados, tampoco se pueden editar.
