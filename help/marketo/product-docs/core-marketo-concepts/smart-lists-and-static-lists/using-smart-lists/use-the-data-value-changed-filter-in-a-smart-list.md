# Utilizar el filtro de valores de datos modificados en una lista inteligente {#use-the-data-value-changed-filter-in-a-smart-list}

Los valores cambian constantemente y se actualizan en la base de datos de Marketo. A veces, en lugar de buscar un valor específico, desea buscar personas que hayan cambiado de valor. Así es como.

>[!PREREQUISITES]
>
>[Crear una lista inteligente](/help/marketo/product-docs/core-marketo-concepts/smart-lists-and-static-lists/creating-a-smart-list/create-a-smart-list.md)

1. Vaya a la **Actividades de marketing** .

   ![](assets/ma.png)

1. Seleccione la lista inteligente y haga clic en el botón **Lista inteligente** pestaña .

   ![](assets/two-1.png)

1. Busque la **Valor de datos cambiado** filtre y arrástrela al lienzo.

   ![](assets/three-1.png)

1. Establezca **Atributo** al campo en el que desea buscar los cambios de valor de datos.

   ![](assets/four.png)

Este filtro buscará todas las personas que hayan cambiado (cualquier cambio) su estado. Si quieres ser aún más rico, intenta [adición de una restricción a un filtro de lista inteligente](/help/marketo/product-docs/core-marketo-concepts/smart-lists-and-static-lists/using-smart-lists/add-a-constraint-to-a-smart-list-filter.md).
