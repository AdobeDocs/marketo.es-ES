# Cancelar una ejecución programada de campaña por lotes {#cancel-a-scheduled-batch-campaign-run}

Si ha programado una ejecución para una de las campañas por lotes y desea cancelarla, así es como.

1. Seleccione la campaña y vaya a la pestaña **Schedule** .

   ![](assets/image2014-9-22-16-3a43-3a10.png)

1. Busque la ejecución programada que desea cancelar y haga clic en el **x** situado junto a ella.

   ![](assets/image2014-9-22-16-3a43-3a15.png)

1. Confirme la cancelación haciendo clic en **OK**.

   ![](assets/image2014-9-22-16-3a43-3a24.png)

>[!NOTE]
>
>Hacer esto no ayudará si la ejecución ya ha comenzado. Utilícelo cuando decida que una ejecución futura debe cancelarse.
