# Añadir a Programa de compromiso {#add-to-engagement-program}

## Información general {#overview}

La campaña inteligente que genera con este paso de flujo será la puerta de enlace a su programa de participación.

![](assets/image2014-9-22-14-3a47-3a32.png)

## Uso {#usage}

1. Seleccione el programa de participación al que desee agregar las personas.

   ![](assets/image2014-9-22-14-3a47-3a36.png)

1. Seleccione la secuencia en la que desee colocar a las personas.

   ![](assets/image2014-9-22-14-3a47-3a39.png)

   >[!NOTE]
   >
   >No se puede agregar una persona a varios flujos dentro del mismo programa.
