# Búsqueda de campañas web que utilicen un segmento específico {#find-web-campaigns-that-are-using-a-specific-segment}

¿Busca campañas web que utilicen un segmento específico?

1. Vaya a **[!UICONTROL Segmentos]**.

   ![](assets/new-dropdown-segments-hand-1.jpg)

1. Busque un **segmento**. Seleccione **Nombre del segmento**. En el panel derecho, haga clic en **[!UICONTROL Campañas asociadas]** para ver las campañas asociadas a este segmento específico.

   ![](assets/image2014-11-26-14-21-59.png)

1. Ver las **campañas** asociadas con el segmento seleccionado.

   ![](assets/image2014-11-26-14-3a25-3a30.png)
