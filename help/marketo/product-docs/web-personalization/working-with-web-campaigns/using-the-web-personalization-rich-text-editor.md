# Uso del Editor de texto enriquecido de personalización web {#using-the-web-personalization-rich-text-editor}

![](assets/one.png)

El editor de texto enriquecido de Personalización web que se encuentra en la página Establecer campañas permite aplicar formato de texto, vincular e insertar imágenes. Incluye opciones de formato similares a los procesadores de texto estándar.
