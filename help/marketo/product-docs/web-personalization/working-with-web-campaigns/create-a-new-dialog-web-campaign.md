# Crear una nueva campaña web de cuadro de diálogo {#create-a-new-dialog-web-campaign}

## Creación de una campaña web de cuadro de diálogo {#create-a-dialog-web-campaign}

Cree una campaña web para personalizar el contenido web en tiempo real y proporcione el mensaje correcto al usuario correcto en el momento adecuado.

Una campaña web es una reacción personalizada asociada a un segmento específico. La reacción puede ser un cuadro de diálogo en el sitio web, y [en sustitución de zona](/help/marketo/product-docs/web-personalization/working-with-web-campaigns/create-a-new-in-zone-web-campaign.md), [función widget](/help/marketo/product-docs/web-personalization/working-with-web-campaigns/create-a-new-widget-web-campaign.md) o una alerta por correo electrónico.

1. Vaya a **Campañas web**.

   ![](assets/image2016-8-18-15-3a48-3a45.png)

1. Select **Crear nueva campaña web**.

   ![](assets/image2016-11-4-10-3a58-3a32.png)

1. Seleccione el **Diálogo** tipo de campaña web. Diseñe y añada el elemento creativo en el editor. Haga clic en **Vista previa** para ver cómo reaccionará la campaña web en el sitio.

   ![](assets/new-3.png)

## Editar una campaña web {#edit-a-web-campaign}

1. En el **Campañas web** página, haga clic en **Editar** en la campaña web.

![](assets/image2016-11-4-11-3a6-3a19.png)

>[!NOTE]
>
>Para facilitar la búsqueda de la campaña web que desea, use la variable [función de filtro](/help/marketo/product-docs/web-personalization/working-with-web-campaigns/filter-web-campaigns.md).

## Vista previa de una campaña web {#preview-a-web-campaign}

1. En el **Campañas web** página, haga clic en **Vista previa** en la campaña web que desee ver.

![](assets/image2016-11-4-11-3a8-3a58.png)

## Clonar una campaña web {#clone-a-web-campaign}

Consulte [Clonar una campaña web](/help/marketo/product-docs/web-personalization/working-with-web-campaigns/clone-a-web-campaign.md).

## Eliminar una campaña web {#delete-a-web-campaign}

1. En el **Campañas web** página, haga clic en **Eliminar** en la campaña web que desee eliminar.

   ![](assets/web-campaigns-1-delete-hand.png)

1. Aparece un mensaje de confirmación para confirmar si desea eliminar la campaña web.
