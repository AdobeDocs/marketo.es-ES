# Archivar una campaña web {#archive-a-web-campaign}

1. Vaya a **Campañas web**.

   ![](assets/one.jpg)

1. Haga clic en la lista desplegable de estado de la campaña web deseada y seleccione **Archivo**.

   ![](assets/two-3.png)

   >[!NOTE]
   >
   >Las campañas web archivadas no se mostrarán en el filtro predeterminado. Para verlos, haga clic en el icono Filtro y en la parte inferior **Estado**, seleccione **Archivado** casilla de verificación y haga clic en **Aplicar**.
