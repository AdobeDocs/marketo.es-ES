# Habilitar/deshabilitar suscripciones a informes {#enable-disable-report-subscriptions}

La personalización web tiene varios informes útiles que se envían por correo electrónico. A continuación se indica cómo suscribirse a ellos.

1. Inicie sesión en Web Personalization. En su nombre de inicio de sesión, haga clic en **Configuración de usuario**.

   ![](assets/image2014-9-17-20-3a48-3a28.png)

1. Seleccione el informe al que desee suscribirse y su frecuencia. Clic **Guardar**.

   ![](assets/email-settings.png)

   ¡Eso es todo! Para cancelar la suscripción al informe, simplemente desactive y haga clic en **Guardar**.
