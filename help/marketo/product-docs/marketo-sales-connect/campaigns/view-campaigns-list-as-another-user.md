# Lista de Campañas de vista como otro usuario {#view-campaigns-list-as-another-user}

Como administrador, puede vista de campañas como cualquier usuario.

>[!NOTE]
>
>**Se requieren permisos de administración**

1. En la aplicación Web, haga clic en **Campañas**.

   ![](assets/one-5.png)

1. Haga clic en la lista desplegable **Vista como** y seleccione el usuario que desee.

   ![](assets/two-4.png)

1. Ahora está viendo campañas como el usuario seleccionado.

   ![](assets/three-4.png)

   >[!NOTE]
   >
   >También puede utilizar filtros o la función de búsqueda junto con Vista En cuanto a la vista de lo que sea más relevante para usted.
