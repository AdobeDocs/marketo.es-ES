# Editar una categoría de plantilla {#edit-a-template-category}

No se puede eliminar una categoría completa ni mover plantillas de forma masiva a una nueva categoría en este momento.

Lo que puede hacer es editar cada plantilla dentro de la categoría para un nombre de categoría existente o uno nuevo. Esto moverá todas las plantillas existentes a una carpeta diferente. Una vez que la carpeta esté vacía, desaparecerá.

Estamos trabajando para agregar la capacidad de hacerlo en masa.
