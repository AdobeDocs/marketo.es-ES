# Editar una Categoría de plantilla {#edit-a-template-category}

No se puede eliminar una categoría completa ni mover las plantillas de forma masiva a una nueva categoría en este momento.

Lo que puede hacer es editar cada plantilla de la Categoría con un nombre de Categoría existente o un nombre nuevo. Esto moverá todas las plantillas existentes a una carpeta diferente. Una vez que la carpeta esté vacía, desaparecerá.

Estamos trabajando para agregar la capacidad de hacerlo en masa.
