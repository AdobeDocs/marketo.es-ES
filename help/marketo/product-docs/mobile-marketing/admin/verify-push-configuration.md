# Verificar configuración push {#verify-push-configuration}

>[!PREREQUISITES]
>
>El desarrollador de móviles debe haber añadido el código SDK de Marketo a la aplicación móvil.

Compruebe la configuración de push para asegurarse de que todo está sincronizado correctamente.

1. Haga clic en **Administrador**.

   ![](assets/image2015-4-22-16-3a12-3a32.png)

1. Select **Aplicaciones móviles**.

   ![](assets/image2015-4-22-16-3a14-3a29.png)

1. Seleccione la aplicación móvil que desee.

   ![](assets/image2015-4-22-16-3a33-3a19.png)

1. Haga clic en **Acciones de aplicación móvil** y seleccione **Verificar configuración push**.

   ![](assets/image2015-4-22-17-3a25-3a8.png)

1. Confirme que las plataformas deseadas están leídas y verificadas y haga clic en **Cerrar**.

   ![](assets/image2015-4-22-18-3a52-3a38.png)

¡Todo listo! Ahora puede empezar a enviar notificaciones push.
