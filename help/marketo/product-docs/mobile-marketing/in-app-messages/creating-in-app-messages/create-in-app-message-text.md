# Crear texto de mensaje en la aplicación {#create-in-app-message-text}

Haga clic en el área de texto para trabajar con el estilo y el contenido del texto.

1. Haga clic en Texto principal para editar en línea.

   ![](assets/image2016-5-6-9-3a56-3a56.png)

1. Para elegir el color del texto, haga clic en él o introduzca los números hexadecimales o RGB en el selector de color.

   ![](assets/image2016-5-6-9-3a59-3a1.png)

1. Haga clic en las flechas para seleccionar un tamaño de texto.

   ![](assets/image2016-5-6-10-3a6-3a51.png)

   >[!CAUTION]
   >
   >Un tamaño de texto demasiado grande puede empujar el contenido más allá de las tres líneas visibles de texto como máximo.

1. Elija un estilo de tipo opcional, para resaltar: **Negrita**, _Cursiva_ o Subrayado.

   ![](assets/image2016-5-6-10-3a15-3a32.png)

1. Elija la alineación del texto: Izquierda, Centrado, Derecha. Centrado es el valor predeterminado.

   ![](assets/image2016-5-6-10-3a18-3a45.png)

   >[!NOTE]
   >
   >La fuente de texto que se representa en el anuncio es la estándar para cada plataforma: Helvetica para Apple y Roboto para Android

1. Marque la casilla para seleccionar la acción Tocar texto .

   ![](assets/image2016-5-6-10-3a20-3a41.png)

1. Seleccione la acción de pulsar para cada plataforma: Apple o Android.

   ![](assets/image2016-5-6-10-3a22-3a12.png)

   >[!NOTE]
   >
   >Para las acciones de punteado, puede configurar diferentes acciones para las plataformas Apple y Android. Por ejemplo, los vínculos profundos se gestionan de forma diferente en Apple y Android. Si el mensaje va solo a una plataforma o a otra, puede dejar la otra en la configuración predeterminada o seleccionar Ninguna.

1. Haga clic en el Texto de apoyo para editarlo en línea. Funciona del mismo modo que editar el texto principal, pero el tamaño predeterminado del texto es menor.

   ![](assets/image2016-5-6-10-3a26-3a27.png)

1. Para el texto principal o de soporte, haga clic en el icono Token para agregar un token.

   ![](assets/image2016-5-6-10-3a29-3a2.png)

1. Seleccione un Token en la lista desplegable, añada un valor predeterminado y haga clic en **Insertar**.

   ![](assets/mytoken.png)

   >[!NOTE]
   >
   >Solo Mis tokens están disponibles como opciones. Si no se ha configurado ningún token en el programa, la lista desplegable Token estará vacía.

   >[!TIP]
   >
   >Asegúrese de tener en cuenta la longitud de caracteres a los que se resolverá un token cuando lo vea la audiencia. Deje espacio suficiente para tener en cuenta valores potencialmente más largos para evitar que se corten.

   Los cambios que realice en los tokens utilizados en un programa de mensajes en la aplicación aprobado no tendrán efecto en el mensaje en la aplicación hasta que el programa se ponga en pausa y se reanude.

Eres dorado. A continuación, debe [configuración del botón de mensaje en la aplicación](/help/marketo/product-docs/mobile-marketing/in-app-messages/creating-in-app-messages/set-up-the-in-app-message-button.md).
