# Investigar posible cliente {#investigate-lead}

Descubra por qué alguien no cumplió los requisitos de MQL, calificó para un programa o progresó a través del ciclo vital.

Para ver la lista completa de escenarios de investigación, haga clic en la flecha hacia abajo a la derecha.

CAPTURA DE PANTALLA

| **Título** | Descripción |
|----------|------------|
| **Investigación de MQL** | Proporciona detalles sobre por qué un posible cliente específico no calificó, dejó de interactuar o se quedó atascado. |
| **Calificación del programa** | Proporciona detalles sobre por qué un posible cliente no cumplió los requisitos para un programa de nutrición. |
| **Progresión del ciclo vital** | Se utilizará para explorar por qué un posible cliente está atascado en una fase más larga de lo esperado. |
| **Análisis de participación** | Para usar cuando un posible cliente ha dejado de abrir o hacer clic en correos electrónicos. |
| **Recorrido de campaña** | Se utilizará para ver la ruta completa y los puntos de contacto de un posible cliente en el Recorrido. |
| **Calidad de datos** | Para usar cuando haya una posible expectativa de que los datos incorrectos están causando un problema con el posible cliente. |
| **Entrega de ventas** | Para usar cuando Ventas rechazó un posible cliente y no realizó el seguimiento. |
| **Atribución** | Se utilizará para explorar qué ha influido en un posible cliente que ha llevado a generar una oportunidad. |
| **Deterioro de puntuación** | Para investigar por qué una puntuación de posible cliente cayó inesperadamente. |

>[!TIP]
>
>Para ver rápidamente una lista de los nueve escenarios, escriba `/lead investigation` en la ventana de solicitud de la parte superior.
