# Añadir SSL a las páginas de aterrizaje {#add-ssl-to-your-landing-pages}

El cifrado SSL (Secure Socket Layer) le permite proteger todas las páginas de aterrizaje para una instancia de Marketo.

Al rellenar un formulario web o visitar una página de aterrizaje alojada en Marketo, la información se envía de forma predeterminada a través de un protocolo no seguro (HTTP). Según la política de su empresa, es posible que desee proteger la información enviada a Marketo a través de (HTTPS). Por ejemplo, cuando visite `https://info.mydomain.com/`, ahora será `https://info.mydomain.com/`.

Marketo rastrea &quot;Página web visitada&quot; y &quot;Click Link on Web Page&quot; de forma predeterminada sobre el protocolo HTTP no seguro. Si desea que los vínculos de seguimiento estén protegidos con su propio certificado, debe tener Marketo para crear un servidor no compartido independiente que lo habilite. Para proteger todos los aspectos de la interacción de un contacto con usted, normalmente significa proteger las páginas de aterrizaje y los vínculos de seguimiento.

Para ver las preguntas más frecuentes sobre la adición de SSL a sus páginas de aterrizaje, consulte [este artículo](https://nation.marketo.com/t5/Knowledgebase/Overview-amp-FAQ-Secured-Domains/ta-p/300900).

Si desea SSL para sus páginas de aterrizaje de Marketo, póngase en contacto con su representante de ventas de Marketo.
