# Explicación de los elementos y las variables en las plantillas guiadas {#understanding-elements-and-variables-in-guided-templates}

Las plantillas de página de aterrizaje guiada tienen dos tipos de secciones editables: elementos y variables.

## Elementos {#elements}

Los elementos son las diferentes partes de contenido que conforman una página de aterrizaje. Pueden ser imágenes, texto o recursos de marketing.

![](assets/image2015-5-20-14-3a57-3a55.png)

Cuando edita una página de aterrizaje guiada, los elementos se mostrarán si se han marcado como editables en la plantilla. Los elementos tendrán los iconos siguientes:

* ![—](assets/image2015-5-20-12-3a30-3a48.png) Imagen
* ![—Formulario ](assets/image2015-5-20-12-3a31-3a33.png)de marketing
* ![—](assets/image2015-5-20-12-3a41-3a21.png)Texto
* ![—](assets/image2015-5-20-12-3a42-3a47.png)Video
* ![—Botón ](assets/image2015-5-20-12-3a44-3a17.png)Compartir de marketing
* ![—Encuesta ](assets/image2015-5-20-12-3a43-3a21.png)de marketing
* ![—Referencia de ](assets/image2015-5-20-12-3a43-3a2.png)marketing
* ![—Apuestas ](assets/image2015-5-20-12-3a44-3a40.png)de marketing
* ![—Fragmento ](assets/image2015-5-20-12-3a47-3a45.png)de marketing

Variables

Las variables son atributos de tipo token que se pueden personalizar desde el editor de páginas de aterrizaje guiadas, como se muestra a continuación.

![](assets/image2015-5-20-15-3a0-3a2.png)

Existen tres tipos de variables: variables de cadena, variables de color y variables booleanas.
