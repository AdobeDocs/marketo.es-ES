# Glosario de actividades de las acciones de perspectiva de ventas {#sales-insight-actions-activity-glossary}

En las acciones de perspectiva de ventas, cuando un vendedor: añade un posible cliente a una campaña de ventas, le envía un correo electrónico de ventas o realiza una llamada de ventas saliente; se registrará en el historial de actividades de Marketo para ese posible cliente. Además, cuando el posible cliente interactúa con correos electrónicos, aperturas, clics y respuestas, también se registra.

Las siguientes actividades se registrarán en Marketo desde las acciones de perspectiva de ventas.

>[!NOTE]
>
>Estas actividades y atributos están disponibles para su consumo desde nuestras API de REST y Bulk.
