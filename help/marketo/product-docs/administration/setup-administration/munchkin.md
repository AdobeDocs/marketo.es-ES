# Munchkin {#munchkin}

Munchkin es el fragmento JavaScript que usted pone en su sitio web. Rastrea visitas, clics y rellenos de formulario. También tiene una API rica para hacer todo tipo de trucos.

Consulte la [página del desarrollador en Munchkin](https://developers.marketo.com/documentation/websites/lead-tracking-munchkin-js/).
