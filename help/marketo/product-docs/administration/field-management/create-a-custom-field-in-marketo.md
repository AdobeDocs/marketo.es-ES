# Crear un campo personalizado en Marketo {#create-a-custom-field-in-marketo}

Si necesita un nuevo campo personalizado en Marketo para almacenar/capturar datos, así es como crearlos.

1. Vaya a Administración y haga clic en **Administración de campos**.

   ![](assets/image2014-9-24-13-3a46-3a26.png)

   >[!TIP]
   >
   >Si desea que los campos se mantengan sincronizados con su CRM, créelos en CRM y se crearán automáticamente en Marketo.

1. Haga clic en **Nuevo campo personalizado**.

   ![](assets/two.png)

1. Elija el tipo de campo. Esto cambiará el modo en que se procesa en listas inteligentes y formularios en Marketo.

   >[!TIP]
   >
   >Consulte el [Glosario de tipos de campos personalizados](/help/marketo/product-docs/administration/field-management/custom-field-type-glossary.md).

   ![](assets/image2014-9-24-13-3a47-3a42.png)

1. Introduzca el Nombre tal como desea que aparezca en Marketo. El nombre de la API se genera automáticamente. Puede modificarlo, pero no se le puede cambiar el nombre una vez establecido. Haga clic en **Crear** cuando termine.

>[!CAUTION]
>
>Los nombres de campo no pueden comenzar con los siguientes caracteres: **. &amp; +[]**

![](assets/image2014-9-24-13-3a48-3a26.png)

>[!NOTE]
>
>El nombre de la API lo utilizan la API SOAP y otros procesos backend.

Ahora puede utilizar este campo personalizado en formularios, pasos de flujo y listas inteligentes.
