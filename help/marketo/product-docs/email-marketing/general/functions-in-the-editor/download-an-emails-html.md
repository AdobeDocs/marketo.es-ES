# Descargar el HTML de un correo electrónico {#download-an-emails-html}

Para realizar copias de seguridad y otros fines, Marketo le permite descargar el contenido de HTML de un correo electrónico.

1. Busque el correo electrónico y selecciónelo. En el menú desplegable **[!UICONTROL Acciones de correo electrónico]**, haga clic en **[!UICONTROL Descargar HTML]**.

   ![](assets/one-4.png)

¡Es así de fácil!
