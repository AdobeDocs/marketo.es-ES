# Usar el Panel de Programa de correo electrónico {#use-the-email-program-dashboard}

Compruebe el rendimiento de sus programas [de](http://docs.marketo.com/display/docs/email+programs) correo electrónico con esta vista de panel.

>[!NOTE]
>
>Si el programa tiene una prueba A/B, consulte el Panel de Programa de [correo electrónico - Vista](../../../../product-docs/email-marketing/email-programs/email-program-actions/email-test-a-b-test/use-the-email-program-dashboard-a-b-test-view.md)de prueba A/B.

![](assets/image2014-9-12-14-3a12-3a56.png)

>[!NOTE]
>
>Todos los datos de esta vista están acumulados (incluye la prueba A/B y el envío final por correo electrónico).

## Enviar correo electrónico {#email-send}

Aquí puede ver cuántos correos electrónicos se enviaron, rebotaron y entregaron.

![](assets/image2014-9-12-14-3a13-3a3.png)

>[!NOTE]
>
>Debido a los estándares de entrega por correo electrónico que están fuera del control de Marketing, las estadísticas de Buncidas y Entregadas son aproximadas, no exactas.

## Aperturas/Clics {#opens-clicks}

Este gráfico muestra la cantidad de correos electrónicos abiertos o en los que se hizo clic durante períodos de tiempo específicos después de ejecutar el programa de correo electrónico.

![](assets/image2014-9-12-14-3a13-3a7.png)

>[!TIP]
>
>Observe cómo el número de aperturas/clics disminuye con el paso del tiempo.

## Resumen - Participación {#summary-engagement}

Esto le muestra la puntuación [general de](../../../../product-docs/email-marketing/drip-nurturing/reports-and-notifications/understanding-the-engagement-score.md)participación.

![](assets/image2014-9-12-14-3a13-3a11.png)

## Resumen - Resto {#summary-rest}

El resto de los datos mostrará las opciones Abrir, Clics, Tasa de clics/aperturas y Cancelar suscripciones.

![](assets/image2014-9-12-14-3a13-3a15.png)

>[!TIP]
>
>La tasa de **cancelación de suscripción** en el ejemplo anterior era tan pequeña que Marketo se acercó para darle una mejor apariencia. El segundo número dentro de la barra se agrega simplemente para la escala.

>[!NOTE]
>
>**Definición**
>
>**Las aperturas **se cuentan cuando el destinatario de correo electrónico descarga las imágenes del correo electrónico, lo que incluye un píxel de seguimiento insertado con el marcador. Si el destinatario vista el correo electrónico pero decide no descargar sus imágenes, no se contará como un archivo abierto. Si las imágenes se cargan en el panel de previsualización del destinatario, normalmente se contará como un archivo abierto, pero variará según el cliente de correo electrónico.
>
>**Haga clic para abrir** mide el porcentaje de correos electrónicos que se abrieron y en los que se hizo clic en un vínculo. Tomamos el número de clics únicos dividido por el número de aperturas únicas y luego multiplicamos por 100 para mostrarlo como porcentaje.

## Actualizar Panel {#refresh-dashboard}

Para ver los datos más actualizados, haga clic en el icono de actualización del panel.

![](assets/refreshicon.png)
