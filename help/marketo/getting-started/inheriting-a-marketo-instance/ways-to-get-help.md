# Formas de obtener ayuda {#ways-to-get-help}

¿Todavía no encuentras lo que necesitas? ¿No sabes lo que no sabes? ¡Conéctate con nosotros! Podemos ayudar.

![—](assets/seo-29.png) Para obtener asistencia técnica con algo que no funcione, póngase en contacto con **[Asistencia al cliente de Marketo](https://nation.marketo.com/t5/Support/ct-p/Support){target="_blank"}**.

![—](assets/seo-30.png) Para conectarse con nuestro equipo de Servicios profesionales, póngase en contacto con **[marketoservices@adobe.com](mailto:marketoservices@adobe.com)**.

![—](assets/education-science-08.png) Para desarrollar sus habilidades con Marketo, visite la **[Marketo Education](https://www.marketo.com/education/){target="_blank"}** página.
